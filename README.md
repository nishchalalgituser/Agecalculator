# Agecalculator

A simple and responsive Age Calculator web application built using HTML, CSS, and JavaScript. The application allows users to enter their birth date and instantly calculate their current age in years.

## Features

* User-friendly interface
* Date of birth selection using a date picker
* Instant age calculation
* Responsive design
* Built with pure HTML, CSS, and JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Use

1. Open the application in a web browser.
2. Select your date of birth.
3. Click the **Calculate Age** button.
4. View your calculated age instantly.

## Project Objectives
